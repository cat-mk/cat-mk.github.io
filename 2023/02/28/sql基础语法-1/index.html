sql基础语法

基础语句

select语句

sql中有很多的比较符，如 =，> 、< 、=、>=、<= 、like、between、in 等

and & or 对数据进行过滤

order by 排序

insert into—向表中插入记录

update更新表中的数据

alter table 更新表

like 在where子句中搜索列中的指定模式

union 合并两个或多个select语句的结果

别名(as)

create table创建表

常用函数

user()

  • 查看当前数据库用户

database()

  • 查看当前数据库的名字

version()

  • 查看当前数据库的版本信息

@@hostname

  • 查看当前计算机的用户名

@@datadir

  • 查看mysql的data目录的绝对路径

@@version_compile_os

  • 查看操作系统位数

load_file()

  • 读取文件

like

  • 类似于等号，可以使用通配符

regexp()

  • 只要有就能匹配，like是等于才匹配

sleep()

  • sleep(x)—-暂停x秒后执行

if

  • if(条件,1,0) 如果条件成立，那么返回1，反之返回0，这里的返回值是可以任意修改的，也可以写表达式

mid

  • mid(a,b,c) 从b位置开始，截取a字符串的c位
  • mid()字符串的下标是从1开始算的

substr

  • substr(a,b,c) 从b位置开始，截取字符串a长度为c的字符串，这个与mid不同的是substr的b和c可以自动取整 substr从1开始计位数

left

  • left(a,b) 从左侧截取a的前b位

length

  • length()判断长度 比如说length(database())=8 判断数据库名的长度是不是等于8

ord ascii

  • ord()返回目标对应的ascii码值 ascii()与ord一致

limit

  • 返回几列 比如limit(1,2)的意思就是从第一列开始返回2列 limit是从0开始计数的

concat

  • 连接一个或多个字符串
  • concat_ws ：与concat基本一致，多了参数分隔符
  • group_concat()：连接每一个字符串默认以逗号分割的

into outfile写文件

  • magic_quotes_gpc=OFF
  • 用户有写权限
  • 不能覆盖已经存在的文件
  • 必须是最后一个查询语句
  • 知道网站的绝对路径

floor

  • 向下取整

extractValue

  • 从目标xml中返回包含所查询的字符串

updatexml

  • 修改xml文档中的内容

exp

  • 相当于自然对数的N次方
  • 自然对数是以常数e为底数的对数，记作lnN（N>0）。
